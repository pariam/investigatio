data BEncodeResponse #

This type denotes a response that has been bencoded.

We wrap BEncode so we can create Servant instances on it without the use of orphans, such as MimeRender.

data Event #

When a client hits the announce url to stop, start, or complete, a torrent, it will include an event queryparam. If it doesn't, or if it's empty, then it is a normal interval-checkin. Otherwise it must be either "started", "stopped", or "completed".
